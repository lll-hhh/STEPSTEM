Abstract

We introduce StepSTEM, a graduate-level multimodal STEM benchmark (283 problems) for evaluating interleaved reasoning chains beyond final-answer accuracy. The benchmark enforces text-image complementarity and provides structured multi-step references with textual points and visual regions.

We also propose a step-level evaluation framework that combines local step alignment, global trace coverage, and multi-reference matching. Experiments show that current MLLMs still have a large gap in genuine cross-modal STEM reasoning.

Methodology (Condensed)

Data Curation: candidate collection, strict filtering for visual dependency, and expert-verified solution structuring with step-level text and bounding-box annotations.

Local Evaluation: textual step matching via attention-rollout alignment; visual step matching via region-aware image similarity.

Global Evaluation: LLM-judge based coverage over whole trace, for both textual key points and visual evidence.

Fusion: integrate local/global as fused process score and take best-over-solutions for multi-path problems.

S_fused = sqrt((w_s * S_local^2 + w_j * S_global^2) / (w_s + w_j))
S_final = max_k S_fused(G_k, P)
