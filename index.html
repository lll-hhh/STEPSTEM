Abstract

StepSTEM is a multimodal STEM benchmark for process-level evaluation of interleaved reasoning trajectories. Beyond final-answer correctness, it evaluates the quality of textual and visual reasoning steps using unified local alignment and global semantic coverage signals.

The benchmark contains 283 problems across engineering, physics, chemistry, biology, mathematics, and other domains, with both text-final and image-final tasks. It supports systematic comparison of unified models, interleaved generation models, and understanding-only models.

Quick Start

  1. Download benchmark files from benchmark/.
  2. Run process scoring with scoring_core/tools/evaluate/score.py.
  3. Run global trajectory judge with whole_trace_judge.py.
  4. Evaluate final answers with final_answer_llm_judge.py.
