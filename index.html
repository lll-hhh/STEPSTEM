Unveiling Fine-Grained Visual Traces:
Evaluating MultiModal Interleaved Reasoning Chains in Multimodal STEM Tasks

Code Dataset Evaluation BibTeX

Abstract

Multimodal large language models (MLLMs) show promising reasoning ability, but existing STEM benchmarks often allow unimodal shortcuts and mostly evaluate final-answer accuracy. We introduce StepSTEM, a graduate-level benchmark of 283 problems across engineering, physics, chemistry, biology, mathematics, and other domains, with strict text-image complementarity and fine-grained process supervision.

We further propose a step-level evaluation framework to score textual and visual intermediate reasoning steps, align predicted trajectories with multiple reference solutions via dynamic programming, and evaluate local and global reasoning quality. Experiments reveal substantial headroom for genuine cross-modal STEM reasoning.
